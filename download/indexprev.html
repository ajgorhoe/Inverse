Instalation instructions

Click the link for desired version and save it to your local disk. In order to unzip the windows version just run a self-extracting archive file in a desired folder. Linux and HP-UNIX versions can be uncompressed by the standard 'uncompress' command. Add a location of the executable file to your $path system variable.

Execute the selected command file by running 'inverse <-options> file' from the command prompt.
