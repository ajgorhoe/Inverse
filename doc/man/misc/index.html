  1. Miscellaneous Utilities

    1. Various File Interpreter's Functions

      1. system { command }, execute

        2. Executes the system's command command. This can be any programme or command that can be executed by the operating system on which the optimisation shell runs. command is a string that can include the command-line arguments. It does not need to be in double quotes. Special character sequences are not allowed in command.

        Programme waits until the execution of command is finished and then the interpretation of the command file goes on.

      2. systempar { command }, executepar

        3. Executes in parallel the system's command command. This can be any programme or command that can be executed by the operating system on which the optimisation shell runs. command is a string that can include the command-line arguments. It does not need to be in double quotes. Special character sequences are not allowed in command.

        The command file interpretation goes on while command is being executed.

        The function is currently available only on UNIX systems.

      3. sleep { numsec }

        4. Delays execution of the programme for numsec seconds. The execution can be delayed only for integer number of seconds.

      4. pwdinfo { }

        5. Prints the information about the current directory of the programme to the standard output and to the programme-s output file (the pre/defined file variable outfile).

        This function currently works only on UNIX systems.
