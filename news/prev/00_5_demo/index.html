May 8, 2000

INVERSE Demo Available for Downloads

  A preliminary demo version of the optimisation shell Inverse has been released. Basic information about the programme can be obtained on Inverse cover pages. Complete technical information together with user documentation is available on Inverse R&D information system. Among the others, Inverse information and support system includes user manuals and other documentation, a list of related publications, a set of demonstrative and training examples, the News and Events notice board, a list of related links, and the user board where users can write about their experience and set questions to user and development community.

  The programme can be downloaded from the "Download" pages.
